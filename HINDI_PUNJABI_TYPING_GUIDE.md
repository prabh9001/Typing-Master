### Keyboard Not Typing?
- Make sure you've switched to the Hindi/Punjabi keyboard using your OS keyboard switcher

- Verify the browser window is focused (click on the typing area)

### Wrong Characters Appearing?
- Confirm you're using the correct keyboard layout (Hindi Devanagari or Punjabi Gurmukhi)
- Some keyboards use different key mappings - try a different layout if available

### Lesson Not Advancing?
- Type all characters exactly as shown, including spaces
- Press the correct keys - the system is case and character sensitive
